The Requirements

What do we need to achieve this?

Server Requirements

To create an efficient server, we need to program a server that has ability to keep data and understand the permissions of each type of person using that server.

The server needs to have databases for keeping the class material and quiz files. It also needs a streaming service for the teacher to do screencast and a messaging protocol for the communication between the tutors and the students. This server also requires a key analyzing system to understand each type of person (teacher, student, tutor) and grant them the rights for protocols they need to use.

Li-Fi Requirements

Li-Fi connections require 3 base elements to work. These are Lamp Driver, LED Transmitter&Receiver and User end Transmitter&Receiver.

A Li-Fi network requires direct light to transmit data. Although mirrors can be used in order to reach users far to source of light or to make small dongles powerful. Our user end dongles will use mirrors to strengthen the beam while keeping it small. The Lamp Driver will convert data strings to light beams and LED will generate the light in a frequency that human eye can't catch(>1MHz). Also our user end dongles will include same principle but it will be more dense in a small volume. We are able to achieve this compactness because of user ends won't deliver same amount of data as our server.
