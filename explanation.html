What's going on?

The Server System

Closed Session Network uses a server system that allows teacher, students and tutors to satisfy their needs for the lab class.

The server in Closed Session Network is capable of keeping the class material, quiz files while allowing the teacher to screencast. It also has the instant messaging system for students to ask questions to the tutors. While connecting to the system, users uses a key provided to them. This key allows the server the understand status of the person that is connecting to the server. This allows server to give permissions to each type of person. Li-Fi Approach

To do the data transfer using the server, we use the Li-Fi approach which has 3 key elements in it.

Li-Fi is a bidirectional, high-speed and fully networked wireless connection method similar to Wi-Fi system that most people use nowadays. It has adventages because of the usage of light. It can be transformed 10,000 times faster than Wi-Fi because of that. It can also work in complement with existing and emerging wireless systems. The key elements in Li-Fi are lamp driver, LED and Receiver&Transmitter Dongle. Lamp driver allows data to be converted to light, LED lamp allows the data to be transferred to the place it belongs and lastly, Receiver & Transmitter Dongle allows a device to recieve a sent data.
